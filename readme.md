# Instalasi
- Ketikan di Command Line/ Terminal
```
composer install
```
- Ubah konfigurasi database dan lainnya di .ENV
- Ketikan di Command Line/ Terminal
```
php artisan migrate
```
- Ketikan di Command Line/ Terminal
```
php artisan db:seed
```
- Login dengan
```
username : admin
password : admin
```
